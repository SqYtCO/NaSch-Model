Rules:

Views:

General

The whole application is divided into several views.
They can be activated by shortcuts and be hidden by 'ESC'.

Game View

The Game View is the main view and shown after startup.
It contains the street with all cars. To navigation

Preferences View

The Preferences View can be shown by the shortcut 'P'. It contains the options to adjust the application.
Here are many options: e.g. customizing the colors or creating own rules.
An explanation of the preferences can be found here.

Help View

The Help View is a extension at the left border of the Game View which can be displayed by the shortcut 'H'.
Here are all necessary information for handling the application.

Tool View

The Tool View is a tool bar on the upper side of the Game View.
It contains tools for saving/opening, generating, navigation and drawing as well as some information like the current size of the field.
The bar can be activated by 'T' if you are in the Game View.

Navigation:

The navigation on the game board is possible with several shortcuts. There are options to move slowly (arrow keys/mousewheel) and fast ([CTRL]+arrow keys) or to zoom ([CTRL]+[+/-]/mousewheel). The position can be reset by the shortcut 'M' or with the button 'Reset Position' which can be found in the Tool View.
