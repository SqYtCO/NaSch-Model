To get started here some basic shortcuts for navigation:
-> [H]: show help; here you can read more about Game Of Life, this application and the navigation!
-> [P]: show preferences; here you can configure the application as you wish
-> [T]: show tools; here you have access to many useful tools
-> [I]: show information; here can be found some information about the current simulation
-> [Esc]: hide view (help/preferences/tools)
